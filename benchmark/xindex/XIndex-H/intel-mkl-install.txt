Intel mkl installation procedure given by this gitlab's README.md does not work anymore since it is expired.

Refer the following link to install lastest version of intel mkl.
https://www.intel.com/content/www/us/en/developer/tools/oneapi/onemkl-download.html?operatingsystem=linux&distributions=aptpackagemanager

If you install intel mkl, there is still no include directory needed for cmake. To solve this problem, it is necessary to install intel-oneapi-mkl-devel package by following command.
$ apt install intel-oneapi-mkl-devel
